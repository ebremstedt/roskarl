# 3.1.30

## New features

- **`DSN.build_db2_string(ssl_cert_path=None)`** — builds a DB2 ODBC connection string from a parsed DSN. Format: `DATABASE=...;HOSTNAME=...;PORT=...;PROTOCOL=TCPIP;UID=...;PWD=...`. When `ssl_cert_path` is provided, appends `Security=SSL;SSLServerCertificate=...`. Raises `ValueError` if `port` or `database` are missing. ([#14](https://github.com/ebremstedt/roskarl/pull/14) by @Koutakis)
